# MOTEL

Methodology for Open Technology Data in Energy Models (MOTEL) is an ETH Domain ORD Program project.

MOTEL provides an open, ontology-ready technology database and workflow for energy system models (ESMs).

## Repository Components

- Documentation website (MkDocs + GitHub Pages)
- Streamlit app for interactive data exploration
- FastAPI backend scaffold
- Data files in CSV/JSON-LD-ready formats
- Ontology files in RDF/OWL (Turtle)
- Reproducible workflows (GitHub Actions / Renku-compatible structure)
