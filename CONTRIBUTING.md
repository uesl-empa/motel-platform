# Contributing

Thank you for your interest in contributing to MOTEL.

## How to Contribute

1. Create an issue describing your proposed change.
2. Fork the repository and create a feature branch.
3. Submit a pull request with a clear summary and rationale.

## Expectations

- Keep data provenance explicit in `data/sources/sources.csv`.
- Follow existing folder conventions.
- Keep docs updated when behavior or structure changes.
- Ensure CI checks pass.

## Code and Data Licensing

- Code contributions are accepted under the MIT License.
- Data and ontology contributions are accepted under CC BY 4.0.
