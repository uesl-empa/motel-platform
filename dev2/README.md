# motel-db

Scaffolded project structure for MOTEL workflow v2.



Key ideas:

the 'raw data' save all the key raw information
the 'harmonised data' label/map the data with existing dataset



workflow:
1. define the structure/format to collect the raw data
2. collect information based on the structure of raw data
3. map the raw data to the clasification dataset
4. create the mapped/harmonised data


NOTE:
 - LLM data extraction: highly follow the structure/format of raw data, required less effort for hoarmonisation.
 - for human user, this provide the structure/format to build and provide inputs
